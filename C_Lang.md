# C Language
- It was developed by ```Denies Ritchie```
- C  language is termed as a **High level assembly language** because, 
1. The control constructs and structured data types found in high level languages. 
2. Bit Manipulation, use of register variables only found in Assembly language. 
- In C, when we reach ```return 0``` stops the program. 
- Pre-processor is the first part of the compiler to run. It takes control instructions from the code such as include another file. Always start with ```#```
- Generally, Program starts from main function and ends at return statement.
